Task 1: Data Report

Mục tiêu
```
Trích xuất dữ liệu từ file report.xlsx và tính toán tổng thành tiền trong khoảng thời gian do người dùng nhập.
```

Cách thực hiện
```
1. Tạo giao diện cho phép người dùng upload file .xlsx.
2. Xử lý dữ liệu trên client (JS) bằng thư viện đọc Excel.
3. Tạo TimePicker cho phép người dùng nhập thời gian bắt đầu và kết thúc, lọc dữ liệu theo khoảng thời gian.
4. Tính tổng doanh thu và hiển thị kết quả.
Kiểm tra: Upload file, nhập thời gian và kiểm tra tổng tiền hiển thị đúng.
```

Hướng dấn thực thi 
```
//đi đến task-1
cd task-1

//cài đặt dependencies 
npm install

//chạy ứng dụng
npm run dev
```
